# Python_API_ExchangeCurrency

## MAIN GOAL OF THIS PROJECT
The main goal of this project was to extract Exchnage Rate and Symbol for selected currencies data by using Python and saved this data as JSON files.

## SHORT DESCRIPTION OF THIS PROJECT
Data class was used to hold data values. Data was extracted by websites using a request library.
Data was written to JSON data format and saved in a folder.

All steps were described in [APIExchangeCurrencySymbol.ipynb](https://github.com/Longwinter93/Python_Projects/blob/main/Python_API_ExchangeCurrency/APIExchangeCurrencySymbol.ipynb)


## FINAL RESULT OF THIS PROJECT
Exchange Rate data and Symbol for selected currencies were successfully extracted and saved as JSON files that are located in a JSON files folder.
